# Anti Nsfw Avatar
- A script that detects nsfw avatars in your discord server.

--------------------------------------------------------

### Step  1:
- Config the **config.json**

### Step  2:
- Put your logs channel ID in **index.js** line 48.

### Step  3:
- Open the terminal, execute the command "**npm install**"

### Step  4:
- Start **index.js**
